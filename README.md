# Основные команды
Присваивание:
```
a := 5
```

Сложение:
```
add some_number 5
```

Вычитание:
```
sub other_number 20
```

Умножение:
```
mul anoter_number 199
```

Целочисленное деление:
```
div num 10
```

Вывод на экран
```
print 50
print some_number
```

Операторы сравнения:
```
a > b
a < b
a == b
```

Условный оператор:
```
if some_number > other_number:
  print 10
```
